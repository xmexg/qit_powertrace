# qit电力统计

我的c++练手项目，用来记录每5分钟全校区全宿舍的剩余电量  
后续可能会加入jsp版当结课作业和数据结构版当结课作业

# 从哪拿到的接口
微信小程序配合[WeChatOpenDevTools](https://github.com/JaveleyQAQ/WeChatOpenDevTools-Python)即可  
顺带一提: 整个微信小程序只通过`JSESSIONID`和`HHMM`标识用户
